Getting Started with Client St0r

Welcome to Client St0r!

Client St0r is a self-hosted IT documentation platform designed for MSPs and IT departments.

Quick Start
  1. Organizations: Create organizations for each client you manage
  2. Users: Add team members and assign them to organizations
  3. Assets: Start tracking IT assets, servers, and equipment
  4. Passwords: Store credentials securely in the encrypted vault
  5. Documentation: Create knowledge base articles and diagrams
Key Concepts
Organizations

Each client gets their own organization with complete data isolation. Staff users can access all organizations.

Users & Roles

Four access levels: Owner, Admin, Editor, and Read-Only. Control access with 42 granular permissions.

Assets

Track all IT equipment with custom fields, attachments, and relationships.

Security

AES-256 encryption, 2FA, breach detection, vulnerability scanning, and audit logging.

Navigation
  • Dashboard: Overview of recent activity and quick actions
  • Quick Add (+): Fast creation menu in top navigation
  • Search: Global search across all content (Ctrl+K)
  • User Menu: Profile, settings, and documentation

# Background Images

This directory contains background images for user profiles.

## Adding Your Own Backgrounds

1. Place image files (JPG, PNG, or WEBP) in this directory
2. Images should be at least 1920x1080 for best quality
3. Keep file sizes reasonable (under 2MB recommended)
4. Users can select "Random from Gallery" to get a random image from this directory

## Recommended Image Types

- Abstract gradients
- Geometric patterns
- Nature scenes (blurred for readability)
- Tech/circuit board patterns  
- Minimalist designs

## Note

All images in this directory are available to all users when they select "Random from Gallery" mode.

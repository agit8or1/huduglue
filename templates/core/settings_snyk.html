Setup Instructions
1. Create Snyk Account
  • Go to snyk.io
  • Sign up with your GitHub account (free for open source)
  • Connect the HuduGlue repository
2. Get API Token
  • Go to Account Settings → General
  • Click to reveal your Auth Token
  • Copy the token and paste it below
3. Get Organization ID (Optional)
  • Go to Settings → General
  • Find your Organization ID in the URL or settings page
  • Format: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
4. Configure GitHub Actions
  • Go to GitHub repository → Settings → Secrets and variables → Actions
  • Add new repository secret: SNYK_TOKEN
  • Paste your Snyk API token as the value
  • The workflow file is already included: .github/workflows/snyk-security.yml

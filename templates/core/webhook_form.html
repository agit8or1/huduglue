Webhook Security
Secret Signing

When you provide a secret key, HuduGlue will sign webhook payloads using HMAC-SHA256. The signature is sent in the X-Webhook-Signature header as sha256=<hex>.

Your receiving endpoint can verify the signature to ensure webhooks are authentic.

Custom Headers

Use custom headers to add authentication tokens or other HTTP headers to webhook requests. Provide a JSON object like: {"Authorization": "Bearer your-token"}

About MagicPlan

MagicPlan is a mobile app for creating floor plans using your device's camera and sensors.

What gets imported:
  • Project information
  • Floor plan dimensions (converted from meters to feet)
  • Room layouts and labels
  • Location details
Export from MagicPlan:
  1. Open your project in MagicPlan
  2. Tap the Share/Export button
  3. Select "JSON" as the export format
  4. Save the file to your device
  5. Upload the file here
Note: MagicPlan exports include measurements in metric units. HuduGlue automatically converts them to feet/square feet.
